Analytics model library for bots created with JBots framework

### Maven Dependency for Consumers
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>jbotstats</artifactId>
	<version>0.2.0</version>
</dependency>
```
